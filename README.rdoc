== README

Simple example of Dropbox SDK integration.  Adds Dropbox authorization and file upload function.

Set the following environment variables:
DROPBOX_APP_KEY
DROPBOX_APP_SECRET

i.e. in your .bashrc file, add
      export DROPBOX_APP_KEY="< YOUR DROPBOX APP KEY >"
      export DROPBOX_APP_SECRET="< YOUR DROPBOX APP SECRET >"
